# stellar-white-level
StellarPay — A modern React dApp that connects Freighter wallet, shows Stellar Testnet balance, and sends XLM with transaction confirmation.
tellarPay — Stellar Testnet dApp StellarPay is a modern React-based decentralized application (dApp) built on the Stellar Testnet. It allows users to connect their Freighter wallet, view XLM balance, fund accounts, and send transactions with real-time confirmation.

Features ✅ Connect & Disconnect Freighter Wallet ✅ Display Stellar Testnet network & public key ✅ Show real-time XLM balance ✅ Fund account using Friendbot ✅ Send XLM payments on Stellar Testnet ✅ Transaction success feedback with hash & explorer link ✅ Clean modern UI with responsive design

⚙️ Setup Instructions (Run Locally) Follow these steps to run the project on your local machine.

1️⃣ Clone the Repository https://github.com/mayurgangarde4141-gif/stellar-white-level

2️⃣ Install Dependencies npm install

3️⃣ Install Freighter Wallet Extension Download and install Freighter Wallet:

https://www.freighter.app/

4️⃣ Start the Development Server npm run dev Open your browser and visit:

http://localhost:5173

How to Use
5- Connect Wallet

Click Connect Freighter and approve access.
6-- Fund Wallet (first time only)

Click Fund with Friendbot to create your testnet account.
7-- Send XLM

Paste destination testnet address
Enter amount (example: 0.1 XLM)
Click Send Transaction
8-- View Transaction

After success, click the explorer link to view the transaction.
Stellar Testnet Explorer
Transactions can be viewed at:

https://stellar.expert/explorer/testnet
🧰 Tech Stack This project is built using the following technologies:

React (Vite) – Frontend framework Stellar SDK – Build and submit Stellar transactions Freighter Wallet API – Connect Stellar wallet and sign transactions Horizon Testnet API – Fetch account balance and submit transactions JavaScript (ES6) – Application logic CSS3 – Wallet UI styling
